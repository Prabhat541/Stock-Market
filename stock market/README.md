## Features
- List of top Indian companies
- Interactive stock chart

- Frontend: HTML, CSS, JS (Chart.js)
- Backend: Python + Flask
- Data: yfinance API

## How to Run
1. Run backend:
```
cd backend
pip install -r requirements.txt
python app.py
```
2. Open frontend/index.html in browser
